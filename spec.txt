Incode: especificacao resumida atual

- Backend vetorial: Amazon S3 Vectors
- Embeddings: Amazon Titan Text Embeddings v2 via Bedrock Runtime
- Cadastro de projetos: arquivo local em REPO_INDEXER_STATE_DIR/projects.json
- Interface humana: CLI com Typer e Rich
- Interface para agentes: servidor MCP com FastMCP

Fluxo principal:

1. setup
2. register_project
3. index_project
4. search_code

Observacoes:

- Cada projeto registrado recebe um indice dedicado no S3 Vectors.
- O chunking preserva simbolos Python quando possivel.
- A busca usa score semantico do S3 Vectors e um bonus lexical local.
- Este arquivo substitui a especificacao antiga baseada em Postgres e pgvector.
